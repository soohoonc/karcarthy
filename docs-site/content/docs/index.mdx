karcarthy coordinates many AI agents while keeping agents, workflows, and
state as ordinary data. The inner model and tool loop still belongs to a
system such as Pydantic AI, Claude Agent SDK, OpenAI Agents SDK, Codex CLI, a
command-line model, or a local mock. karcarthy owns the data-first coordination
layer above those systems.

```clojure
(require '[karcarthy :as k])

(k/defagent researcher "Research the question and cite sources.")
(k/defagent summarizer "Summarize the findings in one sentence.")

(k/run (k/mock-adapter)
       (k/pipe researcher summarizer)
       "what is a monad?")
```

## Why data-first orchestration

Most agent frameworks and SDKs expose agents and workflows as host-language objects. In
karcarthy, the plan is a value:

```clojure
{:karcarthy/type :pipe
 :steps [{:karcarthy/type :agent
          :name "researcher"
          :instructions "Research the question and cite sources."}
         {:karcarthy/type :agent
          :name "summarizer"
          :instructions "Summarize the findings in one sentence."}]}
```

That shape can be printed, diffed, generated, rewritten with `clojure.walk`,
stored as EDN, converted through JSON, or emitted by an agent and parsed without
`eval`.

## What karcarthy provides

| Area | Surface |
| --- | --- |
| Agents | Plain maps tagged with `:karcarthy/type :agent`. |
| Agent SDKs & CLIs | Pydantic AI, Claude Agent SDK/CLI, OpenAI Agents SDK, Codex CLI, command processes, and the local mock adapter. |
| Workflows | `pipe`, `map`, `reduce`, `iterate`, `bind`, and `converse`. |
| Framework mappings | CrewAI, AutoGen, LangGraph, OpenAI specialist routing, and ADK-style helpers compiled to workflow data. |
| State | Agents, workflows, operations, and history as mutable data. |
| Observability | OpenTelemetry spans through `karcarthy.otel/instrument`. |

## Current status

karcarthy is early and not yet published as a stable release. Use it from git,
expect the public vocabulary to stay small, and treat the state docs as the
direction for the default model rather than a separate mode.
