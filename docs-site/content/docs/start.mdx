title: Quickstart

---

This page gets you to a successful local run first. The examples use the mock
adapter, so they do not call a model or require API keys.

## Requirements

- JDK 21 or newer
- Clojure CLI
- A checkout of `soohoonc/karcarthy`

## 1. Build the CLI

From the repository root:

```bash
clojure -T:build uber
```

That writes `target/karcarthy-0.0.2-standalone.jar`. The repo-local
`bin/karcarthy` launcher uses that jar when it exists and falls back to source
while developing.

## 2. Run one agent

```bash
./bin/karcarthy agent echo --instructions "Echo the input." hi
```

Expected output:

```text
[echo] hi
```

By default, command output is plain text. Ask for the full result object when
you need it:

```bash
./bin/karcarthy agent echo --instructions "Echo the input." hi --json
```

## 3. Run a workflow file

`examples/workflows/echo.json` is a workflow value:

```json
{
  "type": "agent",
  "name": "echo",
  "instructions": "Echo the input."
}
```

Run it:

```bash
./bin/karcarthy run examples/workflows/echo.json hi
```

Expected output:

```text
[echo] hi
```

The same CLI also has a machine protocol:

```bash
./bin/karcarthy json < request.json
```

Use `json` from Python, TypeScript, CI jobs, and other non-JVM callers.

## 4. Use the Clojure API

Add karcarthy from git until a stable release is published:

```clojure title="deps.edn"
{:deps {io.github.soohoonc/karcarthy
        {:git/url "https://github.com/soohoonc/karcarthy"
         :git/sha "<commit sha>"}}}
```

Then build and run a workflow:

```clojure
(require '[karcarthy :as k])

(k/defagent researcher
  "Research the question and cite sources.")

(k/defagent summarizer
  "Summarize the findings in one sentence.")

(k/run (k/mock-adapter)
       (k/pipe researcher summarizer)
       "what is a monad?")
```

The result shape is stable across adapters:

```clojure
{:karcarthy/type :result
 :ok? true
 :agent "summarizer"
 :text "..."}
```
