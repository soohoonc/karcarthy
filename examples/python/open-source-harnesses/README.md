# Python open-source harness emulations

Run from the repository root:

```bash
python3 examples/python/open-source-harnesses/swarm/demo.py
python3 examples/python/open-source-harnesses/crewai/demo.py
python3 examples/python/open-source-harnesses/autogen/demo.py
```

These examples use the JSON bridge (`clojure -M -m karcarthy.cli`) and the
offline mock runner.
