# TypeScript open-source harness emulations

Run from the repository root:

```bash
npx tsx examples/typescript/open-source-harnesses/swarm/demo.ts
npx tsx examples/typescript/open-source-harnesses/crewai/demo.ts
npx tsx examples/typescript/open-source-harnesses/autogen/demo.ts
```

These examples use the JSON bridge (`clojure -M -m karcarthy.cli`) and the
offline mock runner.
