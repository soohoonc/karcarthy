karcarthy requires JDK 21 and the Clojure CLI.

## Run the live example

Set `RESPONSES_API_KEY` or `OPENAI_API_KEY`, then run from a checkout:

```bash
clojure -M:examples basic "Explain what an Agent value is."
```

The example constructs an ordinary Agent value, runs it through the live
Responses transport, and prints the model's answer. Use
`KARCARTHY_OPENAI_MODEL` to override the default model.

## Add karcarthy to a project

```clojure
{:deps
 {io.github.soohoonc/karcarthy
  {:git/url "https://github.com/soohoonc/karcarthy"
   :git/sha "<commit-sha>"}}
 :paths ["src"]}
```

Set `OPENAI_API_KEY` to use the built-in Responses transport.

## Define an Agent

Create `src/example.clj`:

```clojure
(ns example
  (:require [karcarthy :as k]))

(k/defagent assistant
  {:model {:transport :responses :id "gpt-5.6"}
   :instructions "Answer clearly and concisely."
   :output string?})
```

Start a REPL and run it:

```bash
clojure -M
```

```clojure
(require '[example :as example] :reload)

(def run (k/run! example/assistant "Why are Lisp macros unusual?"))

(:status run) ;=> :completed
(:output run) ;=> "..."
```

## Develop at the REPL

Agents and Tools are ordinary Clojure values, so they can be inspected and
redefined without a registry or build step:

```clojure
(k/events run)
(k/definition example/assistant)
(k/expansion example/assistant)
```

After editing `src/example.clj`, reload it and run the new definition:

```clojure
(require '[example :as example] :reload)
(k/run! example/assistant "Try the new definition.")
```

From a karcarthy checkout, `clojure -M:examples repl` starts a plain
`clojure.main` REPL with the Basic, Architect, and Coding examples loaded.
Starting it does not require credentials; only live model calls do.

## Give it a Tool

Add this Tool above the Agent:

```clojure
(k/deftool word-count
  {:description "Count the words in text."
   :input {:type "object"
           :properties {"text" {:type "string"}}
           :required ["text"]
           :additionalProperties false}
   :output integer?}
  [{:keys [text]}]
  (count (re-seq #"\S+" text)))
```

Then add it to the Agent:

```clojure
:tools [word-count]
```

On each model turn, karcarthy sends the Tool's name, description, and input
schema. A Tool call is validated, executed, and returned to the model before
the next turn.
