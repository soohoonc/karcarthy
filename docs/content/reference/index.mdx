## Library API

[Library API](/docs/reference/api) lists Agent and Tool forms, Run functions,
Sessions, model configuration, local Tools, MCP, ACP, events, and
contracts.

## Data

[Data](/docs/reference/data) specifies Agent, Tool, Session, Run, event,
limit, and failure values.

## Agent forms

[Agent forms](/docs/reference/agent-forms) explains how a model
writes and runs a new Clojure Agent, plus the direct `compile-agent!` API.

Protocol behavior has dedicated guides: [MCP](/docs/protocols/mcp) for importing remote
Tools, [ACP](/docs/protocols/acp) for serving an Agent, and [Harbor](/docs/guides/harbor) for the
evaluation boundary.
