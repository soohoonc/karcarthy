---
title: Quickstart
description: Run karcarthy, then define and call an Agent.
---

karcarthy requires JDK 21 and the Clojure CLI.

## Run the offline example

From a checkout:

```bash
clojure -M -m karcarthy.demo "hello"
```

The example uses a deterministic fake model, calls a Clojure Tool, and prints:

```text
HELLO
```

No API key or network call is required.

## Add karcarthy to a project

```clojure
{:deps
 {io.github.soohoonc/karcarthy
  {:git/url "https://github.com/soohoonc/karcarthy"
   :git/sha "<commit-sha>"}}
 :paths ["src"]}
```

Set `OPENAI_API_KEY` to use the built-in Responses transport.

## Define an Agent

Create `src/example.clj`:

```clojure
(ns example
  (:require [karcarthy :as k]))

(k/defagent assistant
  {:model {:transport :responses :id "gpt-5.6"}
   :instructions "Answer clearly and concisely."
   :output string?})
```

Start a REPL and run it:

```bash
clojure -M
```

```clojure
(require '[example :as example] :reload)

(def run (k/run! example/assistant "Why are Lisp macros unusual?"))

(:status run) ;=> :completed
(:output run) ;=> "..."
```

## Give it a Tool

Add this Tool above the Agent:

```clojure
(k/deftool word-count
  {:description "Count the words in text."
   :input {:type "object"
           :properties {"text" {:type "string"}}
           :required ["text"]
           :additionalProperties false}
   :output integer?}
  [{:keys [text]}]
  (count (re-seq #"\S+" text)))
```

Then add it to the Agent:

```clojure
:tools [word-count]
```

On each model turn, karcarthy sends the Tool's name, description, and input
schema. A Tool call is validated, executed, and returned to the model before
the next turn.

or [Chat from the REPL](/docs/examples/chat) for a complete interactive example.
