This reference specifies the implemented native kernel. It intentionally
excludes the former Runner protocol, EDN workflow nodes, JSON workflow schema,
and separate dynamic subsystem.

## Pages

| Page | Contents |
| --- | --- |
| [Library API](/docs/reference/api) | Macros and functions for agents, tools, execution, composition, generated code, and observation. |
| [Generated Code](/docs/reference/json) | Reading, expanding, checking, evaluating, invoking, repairing, and recording model-authored Clojure. |
| [Contracts](/docs/reference/schema) | Agent, Tool, Runtime, instructions, context, Session, Run, event, limit, and failure shapes. |

## Namespace

The common API is exposed from one namespace:

```clojure
(require '[karcarthy :as k])
```

Local tools, prompt composition, Responses-hosted web search, the MCP adapter,
and ACP serving are re-exported by the facade. Lower-level provider translation
remains in narrower namespaces.

## Status

The functions on these pages exist on `main`. Durable suspension, resumable ACP
sessions, MCP Streamable HTTP, and observable compaction are documented as
future work where mentioned.
