## Library API

[Library API](/docs/reference/api) lists Agent and Tool forms, execution
functions, Sessions, model configuration, local Tools, MCP, ACP, events, and
contracts.

## Data model

[Data model](/docs/reference/schema) specifies Agent, Tool, Runtime, Session,
Run, event, limit, and failure values.

## Generated Agents

[Generated Agents](/docs/reference/generated-agents) specifies the source
format, evaluation namespace, compilation pipeline, limits, events, and
security boundary for the zero-arity `agent` Tool and `compile-agent!`.
