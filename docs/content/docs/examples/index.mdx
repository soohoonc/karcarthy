## REPL development

The [REPL example](/docs/examples/repl) uses the included deterministic model,
so it runs without a network connection or API key. It covers Agent execution,
Run inspection, source inspection, redefinition, and the correct way to test an
ACP server without giving it the REPL's own standard input.

## Chat from the REPL

The [chat example](/docs/examples/chat) adds a small `read-line` loop and a
Session around `run!`. It keeps the interface in application code while making
conversation history, Run inspection, reset, and exit behavior explicit.

## Multi-Agent review

The [Clojure example](/docs/examples/clojure) builds a coding review system from
local Tools, model-backed reviewer Runs, an editor, and
an optional architect that can create new Agents through the `agent` Tool.

All examples use the public `(require '[karcarthy :as k])` API.
