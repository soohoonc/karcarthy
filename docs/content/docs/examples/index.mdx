## REPL development

The [REPL example](/docs/examples/repl) uses the included deterministic model,
so it runs without a network connection or API key. It covers Agent execution,
Run inspection, Agent definition inspection, and redefinition.

## Chat from the REPL

The [chat example](/docs/examples/chat) adds a small `read-line` loop and a
Session around `run!`. It keeps the interface in application code while making
conversation history, Run inspection, reset, and exit behavior explicit.

## Agent composition

The [Clojure example](/docs/examples/clojure) contrasts independent `run!`
composition with a model-created Agent tree that shares one Run.

## Dynamic Agent proof

The [dynamic Agent example](/docs/examples/dynamic-agents) prints the actual
Agent and program event lineage while a model submits and runs a new Clojure
Agent form. Its companion evaluator scores three generated programs locally;
the Harbor example repeats the search in isolated tasks with verifier rewards
and ATIF trajectories.

All examples use the public `(require '[karcarthy :as k])` API.
