## Two kinds of orchestration

[Agent composition](/docs/examples/composition) shows orchestration written by
a developer with functions, `future`, and `run!`. [Dynamic
Agents](/docs/examples/dynamic-agents) shows a model submitting a new Clojure
Agent form through the same kernel. Both use ordinary Agent values; there is no
separate workflow representation.

## One evaluation, two execution modes

[Hill Climbing](/docs/examples/hill-climbing) runs one candidate search in two
ways. The local mode is fast and deterministic. The Harbor mode repeats it in
isolated tasks and adds verifier rewards, ATIF trajectories, and a viewer.

The [REPL guide](/docs/repl) is a development workflow rather than another
example. Integration details for [MCP](/docs/mcp), [ACP](/docs/acp), and
[Harbor](/docs/harbor) live in their dedicated guides.
